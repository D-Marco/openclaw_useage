# 开发环境安装步骤

## 安装 nodejs/git

```shell
sudo apt update
sudo apt install git -y
curl -fsSL https://deb.nodesource.com/setup_24.x | sudo -E bash -
sudo apt install -y nodejs
```

## 安装 pnpm

```shell
#启用 Corepack，Corepack 是 Nodejs 内置的包管理器版本管理工具，它允许你在不同项目中使用不同版本的包管理器，而不需要全局安装
corepack enable
#安装并激活 pnpm
corepack prepare pnpm@latest --activate
# 验证
pnpm -v
```

## 设置 pnpm 全局目录

```shell
export PNPM_HOME="/home/lane/.local/share/pnpm"
```

将以上内容写入 ~/.bashrc 文件中，使全局生效

```shell
source ~/.bashrc
```

## 下载 openclaw

````shell
#下载openclaw
git clone https://github.com/openclaw/openclaw.git

## 安装 openclaw

```shell
#进入openclaw目录
cd openclaw
#安装依赖（确保环境吸的cmake版本大于3.19）
pnpm install
# auto-installs UI deps on first run
pnpm ui:build
#运行 OpenClaw 的初始化流程 ，选择和确认完后可以关闭命令行窗口
pnpm openclaw onboard --install-daemon
pnpm setup
# 全局生效
mkdir -p ~/.local/bin
ln -sf ~/openclaw/openclaw.mjs ~/.local/bin/openclaw
chmod +x ~/.local/bin/openclaw
echo 'export PATH=$HOME/.local/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
````

## 开发使用

```shell
# Dev loop (auto-reload on TS changes)
pnpm gateway:watch
```

## 网络设置 tailscale

### 安装 tailscale

```shell
curl -fsSL https://tailscale.com/install.sh | sh
```

### 登录tailcale

```shell
sudo tailscale up
```

然后会让你访问一个地址，登录即可。
最后在命令行使用

```shell
tailscale status
```

会看到类似 **100.xx.xx.xx  device-name  username@tailnet**

### 添加反向代理

```shell
sudo tailscale serve --bg 127.0.0.1:18789
```

然后验证：

```shell
tailscale serve status
```

应该能看到

```shell
HTTPS:
https://xxx.tailxxxxx.ts.net
→ 127.0.0.1:18789
```

### 修改openclaw配置

修改文件 ~/.openclaw/openclaw.json
·
将 gateway 中的 相应内容修改如下：

```json
{
  "gateway": {
    "bind": "loopback",

    "tailscale": {
      "mode": "serve"
    },

    "auth": {
      "mode": "token",
      "token": "YOUR_GATEWAY_TOKEN",
      "allowTailscale": true
    }
  }
}
```

### 访问

使用同一账号在要访问的电脑上也安装和登录 tailscale, 然后在浏览器中访问在服务端使用 tailscale serve status 得到的地址即可。

此时会发现需要配对，切换到服务端使用以下命令查看:

```shell
openclaw devices list
```

会看到pending中有request列的值，复制下来执行以下命令：

```shell
 openclaw devices approve <requestId>
```


## 开启关闭

修改配置后，需要重启，通过以下命令重启生效

```shell
#关闭
systemctl --user stop openclaw-gateway.service
#开启
systemctl --user start openclaw-gateway.service
```


## 卸载 gateway+本地数据

```shell
openclaw uninstall
```
