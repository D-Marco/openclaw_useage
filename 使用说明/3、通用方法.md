# 通用方法

## 查看日志

```shell
openclaw logs --follow
```

## agent tools在界面中无法修改

需要修改配置文件中的:

```shell
"profile": "full"
```

除了full，还有messaging、code等

## 启动 Gateway / Runtime

```shell
openclaw gateway start
```

## 检测

```shell
openclaw doctor
```

## 打开 verbose/debug 模式

```shell
DEBUG=openclaw:* openclaw agent ...

```

## 查看都有哪些 agent

```shell
openclaw agents list
```

## 直接运行 Agent

```shell
openclaw agent --agent {agent_name} --message "hello"
```
